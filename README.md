## 📖 Sobre o Repositório

> **My Websites** é um repositório de **coleção de projetos e exercícios práticos** de desenvolvimento web front-end, construídos ao longo dos estudos de HTML5, CSS3 e JavaScript.

Cada pasta representa um projeto ou exercício independente, com foco em aprender e demonstrar técnicas específicas de layout, estilização, responsividade e interatividade — desde páginas institucionais simples até componentes e interfaces mais elaboradas.

## 🚀 Como Visualizar os Projetos

> Por serem projetos **100% estáticos**, não requerem instalação de dependências ou configuração de servidor.

### 📋 Pré-requisitos

| Requisito | Detalhe |
|:----------|:--------|
| **Navegador moderno** | Chrome, Firefox, Edge ou Safari (versões recentes). |
| **Git** | Para clonar o repositório. |
| **Live Server** *(opcional)* | Extensão do VS Code para hot-reload. |

---

### 🔧 Passo a Passo

**1. Clone o repositório:**

```bash
git clone https://github.com/VictorHJesusSantiago/mywebsites.git
cd mywebsites
```

**2. Acesse a pasta do projeto desejado e abra no navegador:**

```bash
# Exemplo para o projeto-01
cd projeto-01

# Opção A: Clique duplo no index.html
# Opção B: Terminal Linux/macOS
open index.html

# Opção B: Terminal Windows
start index.html
```

---

### 🖥️ Com Live Server (VS Code — Recomendado)

```
1. Abra a pasta do repositório no VS Code
2. Instale a extensão "Live Server" (Ritwick Dey)
3. Clique com botão direito em qualquer index.html
4. Selecione "Open with Live Server"
```

| Método | Endereço |
|:-------|:---------|
| 🖥️ **Arquivo local** | `file:///caminho/mywebsites/projeto-01/index.html` |
| 🔴 **Live Server** | `http://localhost:5500` |
| 🌐 **GitHub Pages** | `https://victorhjsantiago.github.io/mywebsites/` |
